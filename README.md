# ying-mall

> 在线商城WebApp

## 功能模块

- 首页：
  - 顶部Navbar组件
  - 幻灯片组件
  - 加载组件
  - 滚动条组件
  - 返回顶部组件
  - 下拉刷新、上拉加载
  - 图片懒加载
  
- 分类页：
  - 顶部组件
  - 侧栏分类
  - 数据获取、显示
  - 客户端缓存
  
- 搜索页：
  - 搜索框组件
  - 热门搜索
  - 历史搜索
  - 关键词搜索结果

## 技术栈

- 框架：Vue2.0
- 网页布局：flex 结合 定位
- 轮播插件：Swiper
- 打包工具：webpack
